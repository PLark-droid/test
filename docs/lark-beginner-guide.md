# 📱 Lark はじめてガイド

> 🌟 **Lark（ラーク）**は、チャット・ビデオ会議・ドキュメント・カレンダーなどが1つになった便利なビジネスツールです。このガイドでは、はじめてLarkを使う方向けに、やさしく手順を解説します。

---

# 👤 Part 1: はじめての方向け（利用者編）

## 📲 Step 1: Larkアプリをインストールしよう

### iPhone をお使いの方 🍎

| 手順 | やること |
|:---:|:---|
| 1️⃣ | ホーム画面から **App Store** を開く |
| 2️⃣ | 🔍 検索バーに「**Lark**」と入力 |
| 3️⃣ | 「**Lark - Team Collaboration**」を見つける |
| 4️⃣ | 「**入手**」ボタンをタップ |
| 5️⃣ | Face ID または パスワードで認証 |
| 6️⃣ | ダウンロード完了後、「**開く**」をタップ |

> 💡 **ポイント**: 青い鳥のアイコンが目印です！

---

### Android をお使いの方 🤖

| 手順 | やること |
|:---:|:---|
| 1️⃣ | ホーム画面から **Google Play ストア** を開く |
| 2️⃣ | 🔍 検索バーに「**Lark**」と入力 |
| 3️⃣ | 「**Lark - Team Collaboration**」を見つける |
| 4️⃣ | 「**インストール**」ボタンをタップ |
| 5️⃣ | インストール完了後、「**開く**」をタップ |

---

## 🔐 Step 2: アカウントを登録しよう

### 新規登録の手順

```
📧 メールアドレスで登録する場合
```

| 手順 | やること |
|:---:|:---|
| 1️⃣ | アプリを開いて「**新規登録**」をタップ |
| 2️⃣ | 📧 **メールアドレス**を入力 |
| 3️⃣ | 届いた **認証コード（6桁）** を入力 |
| 4️⃣ | **パスワード**を設定（8文字以上） |
| 5️⃣ | **名前**（表示名）を入力 |
| 6️⃣ | 🎉 登録完了！ |

```
📱 電話番号で登録する場合
```

| 手順 | やること |
|:---:|:---|
| 1️⃣ | アプリを開いて「**新規登録**」をタップ |
| 2️⃣ | 📱 「電話番号で登録」を選択 |
| 3️⃣ | **国番号**（日本は +81）を確認し、電話番号を入力 |
| 4️⃣ | SMSで届いた **認証コード** を入力 |
| 5️⃣ | **名前**（表示名）を入力 |
| 6️⃣ | 🎉 登録完了！ |

> ⚠️ **注意**: 認証コードは5分以内に入力してください

---

## 👥 Step 3: グループチャットに参加しよう

グループチャットへの参加方法は **3つ** あります：

### 方法①: 招待リンクから参加 🔗

| 手順 | やること |
|:---:|:---|
| 1️⃣ | 送られてきた **招待リンク** をタップ |
| 2️⃣ | Larkアプリが自動で開く |
| 3️⃣ | 「**グループに参加**」をタップ |
| 4️⃣ | ✅ 参加完了！ |

> 💡 リンクをタップするだけでOK！一番かんたんな方法です

---

### 方法②: QRコードをスキャンして参加 📷

| 手順 | やること |
|:---:|:---|
| 1️⃣ | Larkアプリを開く |
| 2️⃣ | 画面右上の「**＋**」マークをタップ |
| 3️⃣ | 「**スキャン**」を選択 |
| 4️⃣ | 📷 カメラでQRコードを読み取る |
| 5️⃣ | 「**グループに参加**」をタップ |
| 6️⃣ | ✅ 参加完了！ |

> 💡 対面で招待されたときに便利な方法です

---

### 方法③: グループカードから参加 💳

| 手順 | やること |
|:---:|:---|
| 1️⃣ | 別のチャットで送られてきた **グループカード** をタップ |
| 2️⃣ | グループの詳細が表示される |
| 3️⃣ | 「**参加**」をタップ |
| 4️⃣ | ✅ 参加完了！ |

---

## 💬 Step 4: チャットを送信しよう

### メッセージを送る手順

| 手順 | やること |
|:---:|:---|
| 1️⃣ | 画面下の「**メッセージ**」タブをタップ |
| 2️⃣ | 送りたい **チャット（相手）** を選択 |
| 3️⃣ | 画面下の **入力欄** をタップ |
| 4️⃣ | ✍️ メッセージを入力 |
| 5️⃣ | 右側の **送信ボタン** ➤ をタップ |

---

### 便利な機能もつかってみよう！ ✨

| 機能 | 操作方法 | 説明 |
|:---:|:---|:---|
| 📷 **写真を送る** | 入力欄の左にある 📷 アイコン | カメラで撮影 or アルバムから選択 |
| 📎 **ファイルを送る** | 入力欄の左にある ＋ マーク | ドキュメントやPDFを添付 |
| 😊 **スタンプ** | 入力欄の右にある 😊 アイコン | かわいいスタンプでリアクション |
| 👍 **リアクション** | メッセージを長押し | いいね！などのリアクションを追加 |
| ↩️ **返信** | メッセージを左にスワイプ | 特定のメッセージに返信 |

---

### メンション（@）を使おう 📣

特定の人に通知を送りたいときは **@（メンション）** を使います：

| 操作 | 結果 |
|:---|:---|
| `@名前` と入力 | その人に通知が届く |
| `@全員` と入力 | グループ全員に通知が届く |

> 💡 **例**: 「@山田さん 明日の会議の件、確認お願いします！」

---

## 🔔 通知設定を確認しよう

大事なメッセージを見逃さないために、通知をオンにしましょう：

### iPhone の場合
```
設定 → Lark → 通知 → 「通知を許可」をオン
```

### Android の場合
```
設定 → アプリ → Lark → 通知 → 「通知を表示」をオン
```

---

## ❓ 困ったときは

| 症状 | 対処法 |
|:---|:---|
| ログインできない | パスワードをリセット or 認証コードを再送信 |
| 通知が届かない | スマホの通知設定を確認 |
| 動作が重い | アプリを再起動 or 最新版にアップデート |

> 📧 解決しない場合は **hello@larksuite.com** にお問い合わせください

# 🔧 Part 2: 管理者向け（グループ作成・招待編）

## 📝 Step 1: グループチャットを作成しよう

### スマホ（モバイル）での作成手順 📱

| 手順 | やること |
|:---:|:---|
| 1️⃣ | Larkアプリを開く |
| 2️⃣ | 画面右上の「**＋**」マークをタップ |
| 3️⃣ | 「**グループを作成**」を選択 |
| 4️⃣ | **グループ名**を入力（例：「プロジェクトA チーム」） |
| 5️⃣ | 📸 グループアイコンを設定（任意） |
| 6️⃣ | **メンバーを追加**（名前で検索 or 連絡先から選択） |
| 7️⃣ | 「**作成**」をタップ |
| 8️⃣ | 🎉 グループ作成完了！ |

---

### パソコン（デスクトップ）での作成手順 💻

| 手順 | やること |
|:---:|:---|
| 1️⃣ | Larkアプリを開く |
| 2️⃣ | 左上の「**＋（新規作成）**」アイコンをクリック |
| 3️⃣ | 「**グループを作成**」を選択 |
| 4️⃣ | **タイプを選択**：|
|  | 　・ 💬 **チャット** - 通常のグループ |
|  | 　・ 📋 **トピック** - テーマ別の整理されたチャット |
|  | 　・ 🔒 **セキュアチャット** - 機密性の高い会話用 |
| 5️⃣ | **グループ名**を入力 |
| 6️⃣ | **メンバーを追加**（検索・部署選択・既存グループから追加） |
| 7️⃣ | 「**作成**」をクリック |

---

### ⚠️ 外部メンバーを招待する場合の注意点

> 🔴 **重要**: 社外の人（外部ユーザー）は **内部グループには参加できません**

社外の人を招待するには「**外部グループ**」を作成する必要があります：

| グループ種類 | 参加できる人 | 見分け方 |
|:---|:---|:---|
| 🏢 **内部グループ** | 社内メンバーのみ | 通常表示 |
| 🌐 **外部グループ** | 社内＋社外メンバー | グループ名の横に「**外部**」タグ |

## 👥 Step 2: 社外の人（外部ユーザー）を招待しよう

### 方法①: 招待リンクで招待 🔗

**【スマホの場合】**

| 手順 | やること |

| 5️⃣ | 「**リンクをコピー**」を選択 |
| 6️⃣ | 📤 LINE、メール、SMSなどで相手に送信 |

**【パソコンの場合】**

| 手順 | やること |
|:---:|:---|
| 1️⃣ | グループチャットを開く |
| 2️⃣ | 右上の「**⚙️ 設定**」アイコンをクリック |
| 3️⃣ | 「**グループを共有**」を選択 |
| 4️⃣ | 「**リンク**」タブを選択 |
| 5️⃣ | 「**コピー**」をクリック |
| 6️⃣ | 📤 メールやチャットで相手に送信 |

## ⚙️ Step 3: グループの権限を設定しよう

グループ作成後、以下の設定を確認しておくと安心です：

### 推奨設定チェックリスト ✅

| 設定項目 | 推奨 | 説明 |
|:---|:---:|:---|
| **メンバーの招待権限** | 🔒 管理者のみ | 不特定の人が勝手に追加されない |
| **グループ共有権限** | 🔒 管理者のみ | リンク/QRが勝手に拡散されない |
| **メッセージ履歴の表示** | ✅ オン | 新メンバーも過去の会話を見れる |
| **@全員の使用権限** | 🔒 管理者のみ | 通知の乱用を防ぐ |

### 設定変更の手順

```
グループ設定 → 権限設定 → 各項目を調整
```

## 💡 管理者向けTips

### 📌 よくある質問

| 質問 | 回答 |
|:---|:---|
| 招待リンクに有効期限はある？ | 設定で期限を指定可能（無期限も可） |
| 招待を取り消したい | リンクを無効化 or メンバーを削除 |
| 誰がいつ参加したか確認したい | グループ設定 → メンバー一覧で確認 |

### 🛡️ セキュリティのポイント

- 📅 **定期的にメンバーを確認** - 不要なメンバーは削除
- 🔗 **招待リンクは必要な時だけ有効化** - 使い終わったら無効化
- 🔒 **機密性の高い内容はセキュアチャットで** - 通常グループと分ける
